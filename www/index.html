Molten.css

A fixed-fluid, full-width web application css grid

Download Github Documentation

Why you should care

Css grids are like coffee shops. They're everywhere. They all do basically the same thing. But, there are only a few that you call home.

Molten.css is different. It's like that coffeeshop that finally gives you padded seats and an outlet at every table. But it's a coffeeshop that has only one beverage. No scones, no muffins, no lattes, only coffee.

Molten.css FINALLY gives you a quick way to create layouts for web applications. I mean, c'mon. Why hasn't this been done. Why does every css grid framework give you tools for websites. I mean, who even builds those anymore? But, seriously, layout those web apps quick-style.


What I'm not

I'm not responsive (yet). I do not have buttons, tables, menus, froms, progress bars, etc. I have decided to modularize myself and perform one thing well, web applications. VERY WELL.

Fixed columns

  
<div class="grid columns split250">
  <aside class="bar">
    <!-- left sidebar (250px) -->
  </aside>
  <section class="content">
    <!-- the content (fluid) -->
  </section>
</div>
